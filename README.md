# Iterasi-Gauss-Seidel-pada-Sistem-Linear-3-Variabel
Implementasi Metode Gauss-Seidel untuk menyelesaikan Sistem Persamaan Linear (SPL) 3 variabel secara iteratif hingga konvergen. Proses dilakukan dengan perhitungan error relatif pada setiap iterasi hingga mencapai batas toleransi 0,01%–0,1% menggunakan pendekatan numerik.
